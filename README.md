## How to Use

1. Open `index.html` in a web browser
2. Ensure all images are placed in the `images/` folder
3. The website uses Tailwind CSS via CDN, so an internet connection is required for styling

## Notes

- The website is responsive and works on all devices
- All doctor images should be in portrait orientation (4:5 aspect ratio recommended)
- Update the image paths in `index.html` if you use different filenames
